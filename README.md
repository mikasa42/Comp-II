# 💻 Computação II - Exercícios em C

Repositório contendo exercícios desenvolvidos durante a disciplina **Computação II**, utilizando a linguagem **C**.

O objetivo deste repositório é reunir as atividades propostas ao longo da disciplina, abordando conceitos fundamentais de programação estruturada, manipulação de memória e estruturas de dados.

---

## 📖 Sobre a Disciplina

Computação II é uma disciplina voltada ao aprofundamento dos conceitos de programação, com foco na linguagem C e no desenvolvimento de soluções mais robustas para problemas computacionais.

Durante a disciplina são explorados tópicos relacionados à organização de programas, gerenciamento de memória e implementação de estruturas de dados básicas.

---

## 🎯 Objetivos

* Praticar programação em C;
* Desenvolver raciocínio lógico e algorítmico;
* Aprender técnicas de modularização de código;
* Compreender o funcionamento de ponteiros e alocação dinâmica;
* Implementar estruturas de dados fundamentais.
